The World Wide Web

A collection of HTML documents made of of files that have Hyper Text Markup Language. The "links" between pages are called hyperlinks that look like this. If this link is clicked on, it will take you to google.

Components of the Web

The major pieces of the web are you, your web browser run by your computer, the Internet, and the servers. Browsers can include programs such as Internet Explorer, Chrome, Safari, and Firefox. The servers are computers designed to store massive amounts of files for the computer to access from the internet.


HTML Elements and Attributes

HTML Elements are used to tell the web browser how to display content web page. These elements can include the following tags:

  • !DOCTYPE - defines the type of document
  • HTML - defines an HTML document
  • title - defines the title of the document. Is usually viewed on your browser toolbar
  • h1-h6 - HTML headings
  • p - paragraph of the document. Makes an invisible box with text in it
  • br - inserts a line break on the page
  • a - an anchor tag, usually used for links on a page
  • img - an image tag for inserting images
  • span - defines a section in a document
  • div - similar to span, defines section in a document

HTML can also include attributes such as images and links represented by following tags:

  • a href="url">link - this creates a hyperlink to the href attribute value url and names it link
  • img src="url" alt="text"> - displays the image from the src attribute value url, if the image is not displayed, the text from the altattribute is shown

One other thing to note about HTML, is that indents represent a new idea in the code

Summary

The world wide web is a collection of HTML documents that have Hyper Text Markup Language. Some documents have links between other documents by using the "a href" attribute. The web uses your computer's browser to access the internet. The internet then accesses files from a server that stores all of the HTML documents and displays them to your browser. Tags and attributes make up the markup language that is HTML. These elements are read by the computer and then translated to display on the web page. These elements can include files, text, images, links, ect.
